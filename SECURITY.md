# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Lykky, please report it responsibly.

**Email:** [security@lykky.co](mailto:security@lykky.co)

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

## What to Expect

- **Acknowledgement** within 48 hours
- **Assessment** within 1 week
- **Resolution** timeline communicated after assessment

## Scope

This policy covers:

- The Lykky web application ([lykky.co](https://lykky.co))
- This GitHub repository and its CI/CD pipeline
- Any APIs or services operated by Lykky

## Out of Scope

- Third-party services we link to
- Social engineering attacks
- Denial of service attacks

## Disclosure

We ask that you:

- Give us reasonable time to fix the issue before public disclosure
- Don't access or modify other users' data
- Act in good faith

We will credit you in the fix (unless you prefer to remain anonymous).
